# Contributing

InkIt is **open source, but not open contribution.** The code is here to read,
build, and fork freely under the Apache-2.0 license. Development is driven by
the Cartesia team.

**We don't accept feature requests or guarantee PR reviews.** You're welcome to
open a pull request, but there's no commitment to review or merge it, and no
response-time guarantee. If you want a change, the reliable path is to fork.

**Found a bug or have feedback?** Use the feedback form:
https://forms.gle/jXNtDsTaLt2rKQ8N9

We read it; we don't promise to act on it.
